My own experience with speedrunning Portal

Hi, I'm MeepOwned13 and I have been speedrunning this game for 5 months now. My love for the game comes from the love of puzzles and also mostly the fact that I love solving things in the unintended way.
If you've ever thought about speedrunning this game I would highly recommend it. Just know it will take a while to learn but there are some awesome tutorials on the internet. My recommendation is just to learn a couple levels at a time and do some runs to see your improvement, that way you won't get demotivated.
The first time I managed to achieve sub 20 minutes felt awesome and it's that rush that I think it's worth playing for. My Personal Best currently stands at a little above 17 and a half minutes and there's still a lot of room for improvement to get to anything close as the top runners.

the cake is a lie

One of the most fun ways for me to run the game is while talking with friends and streaming the runs to them. We are Computer Science students and came up with an idea to use a sorting algorithm called Bogosort which basically shuffles a certain amount of numbers and checks if they are in the right order. The game is that we start the program and a run at the same time and if the algorithm sorts I have to restart. This can happen in anywhere from a minute to 25 which makes it extremely unpredictable. For anyone wanting to try this idea, my program consists of a shuffle algorythm in Python with a timer, ideal list size is around 11-12.
